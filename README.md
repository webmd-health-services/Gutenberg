# System Requirements

* Windows PowerShell 5.1 and .NET 4.6.1+
* PowerShell Core 6+

# Installing

To install globally:

```powershell
Install-Module -Name 'Gutenberg'
Import-Module -Name 'Gutenberg'
```

To install privately:

```powershell
Save-Module -Name 'Gutenberg' -Path '.'
Import-Module -Name '.\Gutenberg'
```
